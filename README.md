Project Overview


Powerpuff Nail Salon is a comprehensive, responsive web application designed as an informational and booking platform for a modern nail and beauty salon. The project successfully integrates advanced front-end features, demonstrating strong proficiency in HTML, CSS, and advanced JavaScript.

Live Demo: https://ssaltuwwa.github.io/powerpuffnails/
GitHub Repository: https://github.com/ssaltuwwa/powerpuffnails


â€¢ Key Features

This project implements the following advanced functionalities:
 Responsive Design: The layout fully adapts to any device screen size (desktop, tablet, and mobile).
 Light/Dark Mode: Theme switching is implemented, with the user's preference saved persistently in localStorage
 User Authentication: Complete user registration (Sign Up) and login (Log In) system. User data is securely stored in localStorage.
 Profile Page: Upon successful login, the user is redirected to a dedicated profile page displaying their name and email.
 Search & Filter: Functionality to search and filter salon services by name. Search results are saved in localStorage.
 Rating System: Users can rate salon services interactively. These ratings are stored and retrieved from localStorage.
 Form Validation: Comprehensive validation is applied to all interactive forms, including checks for emailformat, password length/complexity, and required fields.
 External API Integration: Integration with the Google Maps API on the contact page to display the salon's precise location.
